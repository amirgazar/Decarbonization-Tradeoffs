Prior to 2022-11-17, colorized TIFs had zero values as symbolized as a gray color. In the current version, zero values have been made transparent. Other colors have not been changed.

The color map for LandScan USA is:
        1 - 2: light yellow  rgb(255,255,190)
       3 - 10: medium yellow rgb(255,255,115)
      11 - 25: yellow        rgb(255,255,0)
      26 - 50: orange        rgb(255,170,0)
     51 - 100: vermillion    rgb(230,76,0)
    101 - 250: red           rgb(230,0,0)
  251 - 35000: maroon        rgb(115,0,0)
